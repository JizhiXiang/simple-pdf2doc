# simple-pdf2doc
用python将pdf转换成docx格式

参考来源：https://github.com/python-fan/pdf2word  <br >
作者写的是多线程，多个文件同时转换。 <br >
我调用了他的函数，单个文件转换，适合无基础的童鞋。<br >

记得安装python库

pip install python-docx==0.8.6   <br >
pip install pdfminer3k==1.3.1
